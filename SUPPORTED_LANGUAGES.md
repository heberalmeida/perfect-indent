## How It Works

Perfect Indent uses **block-based indentation detection** which means it works with any language that uses:

- **Braces**: `{ }`
- **Brackets**: `[ ]`
- **Parentheses**: `( )`
- **Keywords**: `if`, `for`, `while`, `function`, `class`, etc.
- **Colons**: Python, YAML, etc.
- **Tags**: HTML, XML, Vue, etc.

## Usage

Simply open any supported file type and press:
- **Ctrl + Shift + I** (Windows/Linux)
- **Cmd + Shift + I** (macOS)

Or run the command: **Perfect Indent: Fix indentation**

---

*Perfect Indent automatically detects the file type and applies the appropriate indentation rules.*

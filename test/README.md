# Test Files for Perfect Indent

This directory contains **40 test files** with intentionally broken indentation to demonstrate Perfect Indent's capabilities across all supported languages.

## How to Test

1. Open any test file in VS Code
2. Press **Ctrl + Shift + I** (Windows/Linux) or **Cmd + Shift + I** (macOS)
3. Or run the command: **Perfect Indent: Fix indentation**
4. Watch as the indentation is automatically corrected!
